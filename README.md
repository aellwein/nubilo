Nubilo
======

What is it?
-----------

*Nubilo* aims to be a private cloud which can host services (a.k.a. apps). 


Example Installation on a Raspberry PI (running Raspbian):
----------------------------------------------------------

* ``sudo apt-get install luarocks luajit libssl-dev``
* ``sudo luarocks install turbo``
